#### 🎉 WikjxWrt-Packages 🎉
*  适用于openwrt 22.03 及以上的分支
 
*  所有插件都为网上收集的开源插件,感谢作者们的付出.

*  常用OpenWrt软件包源码合集，同步上游更新！

*  关于有好的插件请在issues提交

*  再次感谢以上github仓库所有者！
#### 🌈 使用方式 🌈

```bash
 sed -i '$a src-git wikjxwrt https://github.com/wixxm/wikjxwrt-packages' feeds.conf.default
```
